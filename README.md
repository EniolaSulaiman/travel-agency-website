# Enimbi Travel Agency Website  

A simple responsive front-end website for a travel agency. It includes a homepage, about page, and contact page. You can check it out here: [Live demo](https://eniolasulaiman.github.io/travel-agency-website)

## Features  
- Travel destinations showcase   
- Contact form
- Basic filter
- Responsive design
- Over 70 destinations and tours
- Advanced filter

## Tech Stack  
- HTML  
- CSS  
- Vanilla JavaScript
